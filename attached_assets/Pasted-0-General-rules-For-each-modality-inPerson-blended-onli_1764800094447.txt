0. General rules

For each modality (inPerson, blended, online, hyflex) and each dimension
(learningDesign, assessmentFit, learnerProfile, stageFit, feasibility):

Start with 50
Apply +/- adjustments from the relevant inputs
Clamp final dimension value to [0, 100]


Overall score per modality stays as you already have:

overall = 0.4 * learningDesign
        + 0.2 * assessmentFit
        + 0.2 * learnerProfile
        + 0.1 * stageFit
        + 0.1 * feasibility


Remove all existing hard-coded dimension scores (50, 90, 60, 30, etc.).
