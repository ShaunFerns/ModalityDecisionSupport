4. SCORING ENGINE (BACK-END)

Create a function:

score_modality(module_inputs)


It must output scores 0–100 for each modality:

In-Person

Blended

Online

HyFlex

Weighting Model (must match design spec):
Dimension	Weight
Learning Design Fit	40%
Assessment Fit	20%
Learner Profile & Equity	20%
Stage Fit	10%
Feasibility / Resource Fit	10%

HyFlex scoring rules:

Only score HyFlex positively if:

Staff capacity = "comfortable"

Digital access risk is low

Numbers < 100 students OR tool gives a caution

If constraints not met, HyFlex score = 0–30 max

Output JSON:

{
  "in_person": 72,
  "blended": 88,
  "online": 54,
  "hyflex": 41,
  "dimensions": {...}
}
